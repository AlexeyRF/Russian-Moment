# Russian-Moment
Инструменты для жизни в России <br><br>
stop_extermism.pyw - приложения для ориентации и запрета на вход на материалы признанные экстремисткими на территории Российской Федерации, позволяет искать и блокировать доступ к ним на уровне системы<br><br>
certics.pyw - приложение для управления доверенными сертификатами Минцифры<br><br>
extremism_monitor.pyw - приложение отслеживающее экстремизм в тексте и буфере обмена<br><br>
